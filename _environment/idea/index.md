# 插件

## Translation

### 翻译失败

问题描述：

```text
翻译文档失败: 网络连接超时
```

---

解决方案一：

打开`C:\Windows\System32\drivers\etc`下的`hosts`文件,添加一下内容

```text
203.208.40.66 translate.google.com
203.208.40.66 translate.googleapis.com
```
